# Docs

## Operators

|Syntax|Description|
|-|-|
|`a + b`|Addition|
|`a - b`|Subtraction|
|`a * b` `ab`|Multiplication|
|`a / b`|Division|
|`a ^ b`|Expontentiation|
|`f(a)` `f a`|Function Application|
|`\|a\|`|Absolute Value|
|`(a)` `[a]` `{a}`|Brackets|
|`a, b`|Vector/Matrix|
|`a!`|Factorial|
|`-a`|Negation|
|`x => x + 2`|Arrow Function|

## Types of Statements

|Type|Example|Result|
|-|-|-|
|Expression|`3a / f(5)`|Outputs the return value|
|Definition|`y = mx + b`|Saves the variable with the rvalue|
|Command|`:q`|Runs a built-in command|

## Commands

|Command|Description|
|-|-|
|`:q`|Quit|
|`:ld`|List definitions|
